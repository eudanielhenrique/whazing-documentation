## Importação Histórica (Baileys)

Para a API Baileys, é possível importar até **30 dias de mensagens** no momento da leitura do QR Code.

Para a API Plus, o limite é de 100 mensagens por contato, sendo necessário criar um ticket e fazer a importação manual de cada contato.

> **Dica:** Utilize a importação da Baileys primeiro. Após importar, desconecte o canal e conecte na API Plus para manter o histórico.

Para importar na Baileys:
1. Ao ler o QR Code, marque a opção de importar mensagens.
2. Defina a data limite para importação.
